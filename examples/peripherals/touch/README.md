# Touch

This project will introduce how to use `tuyaopen touch` related interfaces to detect touch events.

* Touch Introduction

  Touch sensing is a technology that detects physical contact on a surface. In embedded systems, touch sensors can detect when a user touches specific areas, enabling intuitive user interfaces. This example demonstrates how to initialize touch sensors, register callbacks for touch events, and handle different touch states such as press, release, and long press.

Detailed interface documentation can be found in the [TuyaOS API Documentation](https://developer.tuya.com/cn/docs/iot-device-dev/tuyaos-wind-ide?id=Kbfy6kfuuqqu3#title-12-TuyaOS%20%E6%96%87%E6%A1%A3%E5%AF%BC%E8%88%AA) in Tuya Wind IDE in VS Code.

## Running Results

Each time the touch sensor is pressed or released, the corresponding event will be printed to the console.

```c
*** TOUCH EVENT PRESSD DOWN *** Channel 1
*** TOUCH EVENT RELEASED UP *** Channel 1
```
